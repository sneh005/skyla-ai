Skyla is a simple AI chatbot that i built to improve my full stack development skills. It allows users to ask questions and get real-time responses.
This project help me to understand how frontend and backend work together.

* What it does
  - users can get real-time responses.
  - simple UI with typing effect.

*Challenges:
why building this i faced several errors while connecting the frontend and backend.
i learned a lot while solving these problems.

*Tech stack
FRONTEND: HTML, CSS, Javascript
BACKEND: node.js, express.js
API: Groq SDK

*How to run:
-clone my repository
  https://github.com/sneh005/skyla-ai.git
-Install dependencies:
  npm install
-Run: node server.js
-Open the index.html(directly)
